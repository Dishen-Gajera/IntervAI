InterAI – AI Interview Preparation Platform

InterAI is an AI-powered interview preparation platform built using the MERN stack. It allows users to upload their resumes and receive intelligent, AI-generated interview questions tailored to their profile. The platform supports both technical and HR interview practice, providing real-time feedback to help users improve their performance.

The application follows a SaaS-based model with a credit system, where users can access features based on available credits and purchase more through integrated payment gateways. It also includes secure authentication, a smooth and responsive UI, and full-stack deployment for real-world usage.

    Key Features
        -> Resume Upload (PDF support)
        -> AI-Generated Interview Questions
        -> Technical & HR Interview Practice
        -> Intelligent Feedback System
        -> Credit-Based Access System
        -> Razorpay Payment Integration
        -> Firebase Google Authentication
        -> Smooth UI with animations
        -> Full Stack Deployment

    Tech Stack
        Frontend: React.js
        Backend: Node.js, Express.js
        Database: MongoDB
        Authentication: Firebase
        Payments: Razorpay
        Animation: Framer Motion
        Deployment: Render
